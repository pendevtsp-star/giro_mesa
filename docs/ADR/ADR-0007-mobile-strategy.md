# ADR-0007 - Mobile

## Decisao

Usar PWA responsiva primeiro para garcom, cardapio QR e painel operacional. App nativo com Expo fica para fase posterior.

## Motivo

PWA reduz custo inicial e permite validar fluxo em celulares/tablets rapidamente.

## Consequencias

Algumas capacidades nativas ficam limitadas ate a fase mobile nativa.
